# Security Policy

Если ты нашёл баг, который может:
- ломать клиент при входе на сервер
- вызывать критические краши
- приводить к порче конфигов/данных
- открывать неожиданные сетевые риски

не публикуй полные детали сразу в публичном issue.

Сначала оформи приватное описание проблемы через доступный канал владельца репозитория или создай issue без эксплуатационных деталей.

## Что полезно приложить
- версия Minecraft
- версия Meteor
- версия Fabric Loader
- шаги воспроизведения
- лог / stacktrace
- скриншоты или видео, если проблема визуальная

## Scope

Эта политика касается:
- crash bugs
- опасных mixin-конфликтов
- проблем в сетевых модулях
- ошибок, способных повредить пользовательскую среду
