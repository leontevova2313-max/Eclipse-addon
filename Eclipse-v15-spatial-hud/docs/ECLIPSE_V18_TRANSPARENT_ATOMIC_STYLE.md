# Eclipse v18 — Transparent / compact-icon / industrial warning style

## Что изменено

- Убран плотный фон-затемнитель, GUI теперь опирается на почти полностью прозрачный backdrop.
- Верхняя панель переделана в компактный icon-dock.
- Визуальный язык смещён в сторону тревожной sci-fi индикации: оранжево-красные акценты, угловые рамки, тонкие scan-lines, жёсткие corner brackets.
- Выделенные карточки модулей получают более агрессивный alert-outline и внутренний pulse-bar.

## Дизайн верхних иконок

Иконки сделаны как компактные квадратные chip-кнопки:

- `W` — Workspace
- `S` — Spatial
- `H` — HUD Editor
- `I` — Inspector
- `D / L` — Theme state
- `X` — Close

Принцип дизайна:

- минимальная площадь;
- читаются на маленьком размере;
- одинаковая геометрия;
- активное состояние подсвечивается снизу и по углам;
- общий силуэт напоминает индустриальную системную панель.

## Визуальная идея

Это не копия чужого UI, а собственная стилизация под retro-industrial combat / death-screen language:

- прозрачная поверхность;
- сигнальные оранжево-красные маркеры;
- техно-рамки по углам;
- сканлайны внутри панелей;
- напряжённый "системный" характер интерфейса.
